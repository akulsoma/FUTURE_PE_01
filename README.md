# 🚀 AI Website Copy Generator (Prompt Engineering Project)

## 📌 Overview

This project demonstrates how to use AI tools and structured prompt engineering to generate high-quality, conversion-focused website copy for local businesses.

It includes homepage content, service descriptions, and strong call-to-action (CTA) sections—tailored to different business types.
